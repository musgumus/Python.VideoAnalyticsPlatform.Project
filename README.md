# Python.VideoAnalyticsPlatform.Project
(ENG)This project aims to develop an intelligent video analytics system capable of real-time object detection using the YOLO algorithm. The system processes live video streams from a computer camera, performs object recognition, and presents the results through a web interface. The core components include Python programming language, FastAPI framework, and OpenCV library. 
(TR)Bu proje, YOLO algoritmasını kullanarak gerçek zamanlı nesne tespiti yapabilen akıllı bir video analiz sistemi geliştirmeyi amaçlamaktadır. Sistem, bir bilgisayar kamerasından gelen canlı video akışlarını işler, nesne tanıma gerçekleştirir ve sonuçları bir web arayüzü üzerinden sunar. Temel bileşenler; Python programlama dili, FastAPI çerçevesi (framework) ve OpenCV kütüphanesini içerir.
